# **App Name**: Apartment Spot

## Core Features:

- User Authentication: Implement user registration and login with JWT (JSON Web Token) authentication. Support three roles: Admin, Landlord, and User.
- Landlord Profile Management: Allow landlords to create and manage their profiles with details like name, email, mobile number, address, and profile picture.
- Apartment Listing Creation: Enable landlords to create apartment listings with details such as photos, price, availability date, and any specific conditions.
- Apartment Search and Display: Allow users to search for apartments based on location and display the available apartments in a list or grid format.
- Map Integration: Integrate a map to display apartment locations, similar to the 'shop near me' feature on Google Maps, showing counts and availability.
- User Profile Management: Enable users to create and manage their profiles with basic details and a profile picture.
- AI-Powered Listing Analyzer Tool: Leverage a tool that examines a user's profile and current listings in order to generate recommendations to them for changes or for other similar users or properties that they may want to follow or see. The listing recommendations could suggest that they would get more inquiries with changes like higher quality images, price, amenity, or other field modifications.
- Mobile Number Verification: Implement mobile number verification during the registration process using OTP (One-Time Password).

## Style Guidelines:

- Primary color: A calming blue (#64B5F6) to convey trust and stability.
- Background color: A light gray (#F5F5F5) to provide a clean and modern feel.
- Accent color: An orange (#FFA726) to highlight key CTAs and interactive elements.
- Body and headline font: 'Inter' sans-serif, offering a clean and modern look.
- Use clear and consistent icons for navigation and apartment details.
- Implement a clean, professional layout with a focus on visual hierarchy and ease of use.
- Use subtle animations for transitions and interactive elements to enhance user experience.
